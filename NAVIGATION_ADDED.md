## 🎯 番茄鐘功能說明

### ApplyMode 頁面 (`/apply`)

**流程**:
1. **選擇任務**: 從待處理任務列表中選擇
2. **啟動計時器**: 點擊 "Start" 開始 25 分鐘番茄鐘
3. **工作**: 專注工作直到計時器結束
4. **完成**: 點擊 "Complete" 標記完成
5. **休息**: 系統自動計算休息時間（短休息 5 分鐘 / 長休息 15 分鐘）

**控制**:
- Start - 開始新的番茄鐘
- Pause - 暫停當前會話
- Resume - 恢復暫停的會話
- Complete - 完成當前番茄鐘
- Cancel - 取消當前會話
